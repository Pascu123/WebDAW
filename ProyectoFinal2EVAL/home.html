¿Qué se imparte en 1º de DAM/DAW?

Programación

Usamos Java como lenguaje y Eclipse como IDE.
En esta clase empezamos aprendiendo cosas sencillas como declaraciones de variables u operaciones básicas, pasamos por conocer y usar las clases y los objetos e incluso trabajamos con vectores, matrices y arrays multidimensionales una asignatura de las más completas.

Bases de datos

Usamos Oracle SQL como gestor de bases de datos y SQL Developer como interfaz gráfica para hacer más sencillo nuestro trabajo.

Entornos de desarrollo

En esta asignatura se tocan diferentes palos, como por ejemplo la depuración de código, la gestión de nuestro repositorio en git o la gestión de un proyecto y cómo llevarlo a cabo teniendo en cuneta las especificaciones del cliente, entre muchos otros.

Lenguaje de marcas

Hemos aprendido a crear páginas web con HTML, a darles estilo con css y a hacerlas responsive con bootstrap.

Formación y orientación laboral

Aun siendo la asignatura menos práctica del FP, se trata de una asignatura interesante, el apartado de nóminas o el de contratos pueden serte de mucha utilidad en el futuro.

Sistemas informáricos

Esta asignatura intenta hacernos comprender mejor que es un ordenador y cómo funciona, se aprende sobre redes, conocemos distinos sistemas operativos y trabajamos con ellos y sus funciones, es una asignatura muy importante ya que entender tu herramienta de trabajo seguramente te ayude a trabajar mejor con ella.
