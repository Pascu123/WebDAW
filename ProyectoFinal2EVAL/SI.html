Hardware:

En informática se refiere a las partes físicas, tangibles, de un sistema informático, sus componentes eléctricos, electrónicos y electromecánicos.

Los cables, así como los muebles o cajas, los periféricos de todo tipo, y cualquier otro elemento físico involucrado, componen el hardware o soporte físico; contrariamente, el soporte lógico e intangible es el llamado software.

Durante la primera parte de la asignatura vamos a entender como funcionan los componentes físicos de un ordenador, así como sus partes y periféricos.

Puedes aprender más sobre Hardware pinchando en la imagen o dejar que nosotros le enseñemos. Para ello puede inscribirse para el curso que viene rellenando el siguiente formulario.

Redes:

Aunque nos gustaría dedicarle mucho más tiempo a este apartado, no hay tiempo suficiente durante el curso como para poder explicar redes de una forma completa.

Por tanto, intentamos dar unas pinceladas a este extenso mundo para que los alumnos puedan empezar con una base si en algún momento deciden dedicarse a esta rama.

Se planteará desde un enfoque teórico ya que por motivos de infraestructura y programación, se nos torna imposible gestionarlo de otra forma.

Sistemas operativos:

Antes de trabajar con sistemas operativos específicos, vamos a tratar de forma teórica que es un sistema operativo y como funciona.

Trabajaremos sobre el concepto de licencia y de software libre. Veremos los distintos sistemas operativos que existen dependiendo de sus características y realizaremos alguna practica para asentar los conocimientos.



Un sistema operativo es el conjunto de programas de un sistema informático que gestiona los recursos de hardware y provee servicios a los programas de aplicación de software.
Estos programas se ejecutan en modo privilegiado respecto de los restantes.

Virtualización:


La virtualización utiliza el software para imitar las características del hardware y crear un sistema informático virtual. Esto permite a las organizaciones de TI ejecutar más de un sistema virtual, y múltiples sistemas operativos y aplicaciones, en un solo servidor.

Durante el segundo trimestre trabajaremos con la virtualización a diario y usaremos para ello una herramienta de oracle llamada...

VM Virtual Box:
Se trata de una herramienta muy util en el plano virtualización de sistemas operativos.

Este software está disponible en su página oficial de forma gratuita y nos permite tener varios sistemas operativos ejecutandose dentro del nuestro al que llamaremos huesped. Las utilidades de esta herramientas son muchas, como por ejemplo, puebas de software, facilidades en el trabajo o su uso con fines académicos.

Aprenderemos a usar esta herramienta de forma práctita, usando e instalando en ella varios sistemas operativos de modo manual.

Windows:

Aunque es un sistema operativo al que estamos acostumbrados, siempre se puede aprender un poco más.

Trabajaremos en una máquina virtual de windows 7 para poder realizar los ejercicios sin problemas.
Probaremos distintas aplicaciones y opciones que nos proporciona este sistema operativo.

Además aprenderemos a programar scripts para windows y a ejecutarlos. Interactuaremos con en cmd y mucho más.
