¿Que se imparte en Lenguaje de marcas?

HTML:

Un lenguaje de marcado o lenguaje de marcas es una forma de codificar un documento que, junto con el texto, incorpora etiquetas o marcas que contienen información adicional acerca de la estructura del texto o su presentación.

Usaremos HTML como lenguaje de marcas durante este curso.
Se trata de uno de los lenguajes de marcas más usados y seguro que a los alumnos les será útil durante su trayectoria laboral.

CSS:

CSS, en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcas.
Si el html es la estructura de la casa, css sería la pintura, los muebles, la decoración, etc.
Trabajaremos con este lenguaje y aprenderemos a asociar nuestros css a html y poder crear nuevas páginas mucho mejores.

Bootstrap:

Bootstrap es un framework CSS utilizado en aplicaciones front-end, es decir, en la pantalla de interfaz con el usuario para desarrollar aplicaciones que se adaptan a cualquier dispositivo.
Usaremos estre framework para hacer que nuestras páginas sean responsive.
